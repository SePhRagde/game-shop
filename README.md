**GameShop** es una tienda en línea de videojuegos creada con React y Bootstrap. Permite a los usuarios explorar, agregar al carrito y comprar videojuegos de manera sencilla.

## Características

- Navegación fácil entre categorías de videojuegos.
- Detalles de productos con información y precios.
- Carrito de compras para gestionar productos seleccionados.
- Resumen de compra antes de finalizar.

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/SePhRagde/game-shop.git
   cd game-shop
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia la aplicación:
   ```bash
   npm start
   ```

## Uso

1. Explora diferentes categorías de videojuegos.
2. Haz clic en "Ver Detalle" para más información.
3. Agrega productos al carrito y ajusta la cantidad.
4. Finaliza tu compra completando el formulario.
